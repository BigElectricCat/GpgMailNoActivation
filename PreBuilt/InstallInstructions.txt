Install GPGSuite 2018.4 suite as normal.
(Installer file GPG_Suite-2018.4_2.dmg)

Copy GPGMail_13.mailbundle and GPGMail_14.mailbundle to
/Library/Application\ Support/GPGTools/GPGMail/
Replacing the existing files.
Make sure to unzip files first.


Alternatively open RemoveGPGMailDRM.dmg and drag the files
to the folder shortcut.
